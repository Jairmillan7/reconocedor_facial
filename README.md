# Reconocedor Facial Web

Una aplicación web interactiva que utiliza visión artificial para el reconocimiento de rostros en tiempo real directamente desde el navegador.

##  Descripción
Este proyecto implementa un sistema de reconocimiento facial utilizando herramientas de JavaScript. La interfaz es minimalista y está diseñada para procesar el flujo de video de la cámara web, detectar rostros y mostrar los resultados visualmente.

##  Tecnologías utilizadas
* **JavaScript:** Lógica principal y procesamiento de video.
* **p5.js:** Para la renderización de la interfaz y el manejo del lienzo (canvas).
* **HTML5 & CSS3:** Estructura y diseño responsivo.

##  Estructura del proyecto
* `index.html`: Punto de entrada de la aplicación.
* `sketch.js`: Contiene la lógica del dibujo y la integración con la cámara.
* `style.css`: Estilos visuales del contenedor y el canvas.
